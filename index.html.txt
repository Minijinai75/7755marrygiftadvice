                <div id="content2" class="content-box bg-gray-50 p-5 rounded-lg border border-gray-200 text-gray-700 text-base sm:text-lg leading-relaxed">這是一個讓任何玩家，在任何時間，都能向已婚玩家贈送「果醬」作為結婚賀禮的永久性社交功能。詳細運作方式如下：

1.  **祝福入口**：在遊戲介面新增一個「贈送賀禮」的常駐按鈕。
2.  **指定對象**：點擊按鈕後，彈出一個輸入框，讓贈禮玩家（A玩家）輸入收禮玩家（B玩家）的「介紹碼」。
3.  **系統驗證**：系統後台根據介紹碼，驗證B玩家的身份以及是否處於「已結婚」狀態。
    * 若驗證通過，則進入下一步。
    * 若驗證失敗，則提示「查無此玩家或對方尚未結婚」。
4.  **贈送賀禮**：驗證成功後，A玩家進入贈禮畫面，可選擇固定額度的「果醬」作為賀禮，並附上一段祝福留言。
5.  **永久記錄**：A玩家送出賀禮後，B玩家的個人主頁上，一個名為「祝福紀念牆」的頁面將會永久記錄下來自A玩家的頭像、暱稱、祝福語與收禮日期。</div>

1.  **滿足玩家情感需求**：解決了玩家「想為朋友的喜事獻上祝福，卻苦無管道」的痛點。「祝福不該有時效性」，此功能讓玩家能隨時為朋友補上賀禮，讓心意不再因時間而錯過。
2.  **提升核心資源價值**：讓核心付費資源「果醬」，除了玩家自我使用外，被賦予了全新的、充滿溫度的社交價值。它能溫和地促進果醬在活躍玩家間的善意流通，創造正向的社群循環。
3.  **強化長期留存**：「祝福紀念牆」會成為已婚玩家最珍貴的社交資產之一，是人氣與友誼的勳章。這份獨特的榮譽感，將有效提升玩家對帳號的情感連結，降低流失率。
4.  **開發成本低、可行性高**：本功能可直接活用遊戲現有的「介紹碼」作為玩家的唯一識別碼（Player ID），無需從零打造複雜的搜尋或好友系統。開發重心僅在於前端介面與後端資料記錄，技術上高度可行，具備快速部署的潛力。</div>

    <script>
        // Copy-to-clipboard functionality
        document.querySelectorAll('.copy-btn').forEach(button => {
            button.addEventListener('click', () => {
                const targetId = button.getAttribute('data-target');
                const contentElement = document.getElementById(targetId);
                
                // Use a temporary textarea to preserve formatting
                const textarea = document.createElement('textarea');
                textarea.value = contentElement.innerText;
                document.body.appendChild(textarea);
                textarea.select();
                document.execCommand('copy');
                document.body.removeChild(textarea);

                // Visual feedback for the user
                const originalText = button.innerText;
                button.innerText = '已複製！';
                button.classList.remove('bg-pink-400', 'hover:bg-pink-500');
                button.classList.add('bg-green-500');
                
                setTimeout(() => {
                    button.innerText = originalText;
                    button.classList.remove('bg-green-500');
                    button.classList.add('bg-pink-400', 'hover:bg-pink-500');
                }, 2000);
            });
        });
    </script>
